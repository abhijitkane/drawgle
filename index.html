Welcome to Drawgle!

Drawgle is a simple environment for drawing. Drawgle is written completely in Python. You can use the in-build interactive prompt, or write script files.

Why?

Drawgle was started as a simple way to learn Python and Git. But, I'll be really glad if it turns out to be something more.

Current Commands

Right now, Drawgle only supports the new and line commands. I plan to add a few more modules in the coming weeks.
